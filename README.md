# sbb_qq
게시판으로 spring연습

### 개발환경
- spring boot
- gradle
- thymeleaf
- spring data jpa
- mysql

### 스프링 시큐리티란?
- 스프링 시큐리티 스프링 기반 애플리케이션의 인증과 권한을 담당하는 스프링의 하위 프레임워크이다.
- 인증(Authenticate)은 로그인을 의미
- 권한(Authorize)은 인증된 사용자가 어떤 것을 할수 있는지를 의미!
